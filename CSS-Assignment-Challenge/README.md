# CSS Assignment Challenge Overview

This repository contains a series of CSS exercises organized by folder. Each exercise demonstrates a different concept from the lesson, including styling, cascade, specificity, and color formats.

## Exercise-1
Focus: external CSS and cascade priority.
Completed task: make the paragraph appear green using **styles.css** even though inline and internal styles were present.
Key lesson: **!important** can override other rules, and inline styles will still win over normal rules unless overridden with higher priority.

## Exercise-2
Focus: debugging broken stylesheet links and CSS syntax.
Completed task: identify and explain errors in **index.html** and **design.css**.
Key lesson: correct attribute names like **href** and **rel="stylesheet"**, plus valid CSS syntax with braces, colons, and semicolons.

## Exercise-3
Focus: color formats and styling example content.
Completed task: build a styled table and/or work through color format questions using named colors, hex, RGB, and HSL.
Key lesson: understanding when to use each color notation and how alpha transparency works with **rgba()**.

## Exercise-4
Focus: external page styling for a press release layout.
Completed task: apply all styles in **press.css**`, link Google Fonts, and ensure headings, body text, and layout follow the design requirements.
Key lesson: external CSS is preferred, and line-through decoration on a date line shows how styling can change meaning.

## Exercise-5
Focus: predicting CSS output.
Completed task: explain how different CSS selectors and inheritance rules affect displayed text color, decoration, and font choice.
Key lesson: inline styles and later rules can override other declarations, and removing a rule can reveal fallback styling.

## Exercise-6
Focus: recreate a screenshot design using blank files and external CSS.
Completed task: create **coffee.html** and **coffee.css** with Google Fonts, HSL heading color, named subheading color, RGB paragraph color, and second-paragraph word-spacing.
Key lesson: build from scratch with proper structure, external stylesheet use, and targeted selectors like **p:nth-of-type(2)**.

## Exercise-7
Focus: conceptual understanding.
Completed task: answer written questions about the roles of HTML vs CSS, why external CSS matters, specificity, and the value of multiple color formats.
Key lesson: CSS styles decoration while HTML provides structure, and knowing the reasoning behind styling choices is as important as writing the code.
