## Color Red in Three Other Formats.
 **Color Name: Red<br>
 Hex: #ff0000<br>

 HSL: hsl (0, 100%, 50%)**

 ## Difference between rgb (255, 0, 0) and rgba (255, 0, 0, 0.5)?
 In coding **rgb(255, 0, 0)** stand for **Red, Green and Blue**, each of the three numbers represents an intensity value ranging from 0 to 255. While
**rgba(225,0,0,0.5)** represents the same red color as rgb(255,0,0) but it has a **0.5 transparency** the first three numbers dictate the mix of **Red, Green, Blue**, while the fourth number dictates the transparency.

#### What the a in **rgba** stand for.
It stands for **Alpha** which controls the opacity of the color. Alpha value is always a decimal percentage between 0.0 and 1.0.

## Converting #000000 and #ffffff to RGB format and what colors they are? 
**#000000 = Black**
Its RGB Format is **rgb(0, 0, 0)**

**#ffffff = White**
    Its RGB Format is **rgb(255, 255, 255).
