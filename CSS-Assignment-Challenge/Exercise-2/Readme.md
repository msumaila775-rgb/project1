# The Broken Stylesheet
In this exercise I was tasked to find and fix any errors related to CSS linking or syntax.

## Errors Found in the HTML File and Corrections.

In the first **link**
**scr** is wrong, it must be **href** to load the css file. 

In the second **link**
**rel="style"** is invalid. It should be **rel="stylesheet"** for the font stylesheet to load.

## Errors Found in the design.css File.
In the body **element**:
Missing **opening brace** after **body**. The **body** rule is invalid.
In the **h1 element**:
Missing semicolon after **color: darkblue**
Missing colon after **font-size**
These syntax mistakes makes the **h1** rule invalid.
